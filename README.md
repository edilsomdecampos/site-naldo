# Naldo | Estúdio de Revestimentos 💎

Este projeto une o artesanato técnico de alto padrão à Inteligência Artificial de última geração. Desenvolvido para o especialista Naldo, o site foca em converter visitantes em clientes através de uma experiência visual imersiva.

## 🤖 Engenharia de IA e Visão Computacional
O diferencial deste projeto foi o uso estratégico de IAs para a criação de ativos:
- **Geração de Imagens de Alta Fidelidade:** Texturas de porcelanato e pedra ferro geradas via IA (Gemini), permitindo mostrar detalhes de acabamento impecáveis.
- **Engenharia de Prompt:** Extração do potencial máximo das ferramentas generativas para uma identidade visual exclusiva.

## 🛠️ Tecnologias e Performance
- **Stack:** HTML5, CSS3 e JavaScript.
- **Analytics:** Integração total com Google Analytics 4 (ID: G-9WZ4J49F9M) para monitoramento de leads.
- **Responsividade:** Design Mobile First (otimizado para celulares).
